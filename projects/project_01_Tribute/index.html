Linus Torvalds

Linus Torvalds is the creator and lead developer of the Linux kernel. The first Linux prototype was released to the public in 1991 with the first non-beta version released a few years later. Linus currenlty works fulltime on improving Linux and is sponsored by the Linux Foundation. Linus is also the creator of the Git version control system which he initially created to aid in the development of the Linux kernel.

  • 1991 First Linux kernel prototype released to public and open sourced.
  • 1994 Linux kernel version 1.0 released.
  • 1996 Asteroid 9793 Torvalds named after Linus Torvalds.
  • 2005 Git version control system officially released.
  • 2010 Becomes United States citizen.
  • 2012 Inducted into the Internet Hall of Fame.
